## II. THE HEALTH WORKER STRIKE CASCADE: ANTICOMMUTATION ENTERS PHASE 2

### Definition Refresh: Anticommutation in Pandemic Systems

**Anticommutation** requires that order of operations matters for outcome. Two systems anticommute when:
- Process A followed by Process B yields outcome X
- Process B followed by Process A yields outcome Y ≠ X
- Order matters critically

In pandemic control:
- V_E (viral exponential) anticommutes with I_R (institutional response) when: rapid detection → rapid vaccine design → deployment arrests outbreak
- Commutation (order independent) occurs when: viruses grow undetected → vaccine designed blindly → deployment too late → outcome same regardless of detection speed
